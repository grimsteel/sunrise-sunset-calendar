Sunrise-Sunset Calendar

Generate an iCal file for sunrise and sunset times for a given location.

Note: For best results, use the canonical timezone identifier: Wikipedia.

This will generate an iCal file with 365 days of sunrise/sunset data.

How do I get this into Google Calendar?

Once you have your iCal file, follow these steps:

  1. Create a new calendar to hold the data
    Other calendars > + > Create new calendar
  2. Navigate to the Import/Export screen
    Other calendars > + > Import
  3. Select the iCal file
  4. Choose the calendar you created earlier
  5. Click Import

The import process may take a while. If everything was successful, you should be informed that 730 events were imported! (365 days with an event for sunrise and sunset each day)

Note: For the times to display correctly, set your Google Calendar timezone to the one you specified above.
